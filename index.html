SoftSkill Observer

Outil de médiation par le jeu

Bilans
1

Inscrire le jeune

Remplissez sa fiche avec lui. Demandez-lui de passer le test 16 Personalities sur son téléphone.

2

Lancer la session de jeu

Laissez le jeune jouer librement. Observez sans intervenir : posture, réactions, interactions avec les autres.

3

Remplir les observations

Pour chaque savoir-être, indiquez ce que vous avez observé. Ce n'est pas une note : c'est un repère pour la discussion.

4

Discuter et orienter

Utilisez le bilan pour échanger avec le jeune et l'orienter vers les secteurs qui correspondent à ses forces.

Rappel : Ces observations sont subjectives et contextuelles. Elles ne définissent pas le jeune. Elles servent de base de discussion pour l'accompagner dans son orientation. Vous pouvez les modifier à tout moment.
