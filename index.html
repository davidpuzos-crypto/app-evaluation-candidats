SoftSkill Observer

Outil de médiation · Séances de jeu

Voir les bilans
1

Inscrire le jeune

Remplissez sa fiche avec lui. Proposez-lui le test 16 Personalities sur son téléphone.

2

Lancer la session

Laissez le jeune jouer librement. Observez sans intervenir : posture, réactions, interactions.

3

Saisir les observations

Pour chaque savoir-être, indiquez ce que vous avez observé. C'est un repère, pas une note.

4

Discuter & orienter

Utilisez le bilan pour échanger avec le jeune sur les secteurs qui correspondent à ses forces.

Rappel : Ces observations sont subjectives et contextuelles. Elles ne définissent pas le jeune — elles ouvrent une conversation. Vous pouvez les corriger à tout moment.
